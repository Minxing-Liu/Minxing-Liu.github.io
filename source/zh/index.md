# 中文记录

这里是中文记录入口，用来保留母语写作和思考的空间。

正式公开版本仍以英文站点为准。中文内容默认不放入顶部导航，也不作为官网主版本呈现。

## 工作方式

- 先在 `drafts/zh/` 中用中文自由写草稿。
- 再让 agent 根据中文草稿生成或更新英文官网文章。
- 如果某些中文内容也想发布到这里，可以单独说明。
