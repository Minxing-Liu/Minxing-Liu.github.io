# Minxing-Liu.github.io

Personal website of **Minxing Liu** — PhD student in Computational Neuroscience and tennis enthusiast.

Built with [Hexo](https://hexo.io/).

## About

This site documents my adventures as a PhD student in computational neuroscience and as an avid tennis player. It includes:

- **Research notes** — computational neuroscience concepts, paper summaries, and research reflections
- **Tennis diaries** — match reflections, technique breakdowns, and the mental game of tennis
- **PhD life** — honest reflections on the graduate school journey

## Local Development

```bash
npm install
hexo server
```

Then open [http://localhost:4000](http://localhost:4000).

## Build

```bash
hexo generate
```
